# AlcoholOrGasolineFuel
Projeto em IONIC que calcula automaticamente a melhor opção entre álcool e gasolina com base nos preços dos combustíveis.

![Alcohol or Gasoline Fuel](/assets/screenshot.png)

## Tecnologias Utilizadas

- **Ionic:** Framework para o desenvolvimento de aplicativos móveis multiplataforma.
- **Angular:** Framework de desenvolvimento web usado pelo Ionic.
- **HTML e CSS:** Tecnologias padrão para estruturação e estilização da interface do usuário.

## Como Utilizar

1. Clone o repositório: `git clone https://github.com/1bertojunior/AlcoholOrGasolineFuel.git`.
2. Instale as dependências: `npm install`.
3. Execute o aplicativo: `ionic serve`.

## Funcionalidades

- Calculadora de custo de abastecimento entre álcool e gasolina.
- Interface intuitiva e fácil de usar.
